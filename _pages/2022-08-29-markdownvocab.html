Vocab List

function - A group of related statements that perform a specific task def - The command to define a function parameter - Variables for the values the function needs. Is passed as an argument when the function is called string - A series of readable or usable sentences that the user will use len() - Returns the length (the number of items) of an object list - A “container” that can store any kinds of values

Helpful Commands

  • cd: change directory
  • cat {filename}: concatenate
  • pwd: print working directory
  • git clone {url}: clones GitHub repository
  • ls: list files in a directory
  • cp {filename}: copy
